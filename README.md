# 🎯 BetHub v7

Hub di gestione scommesse personale. Funziona offline come PWA installabile.

## Funzionalità

- **💰 Budget Hub** — totale in tempo reale, log trasferimenti a tendina, Rage Bet Fund (sbloccato sotto €10)
- **1️⃣2️⃣ Masaniello x2** — formula esatta (Tartaglia + quota variabile), stake aggiornato live, modifica esiti, New Start
- **📈 Scalata** — proiezione live, checkpoint con prelievo, modifica esiti
- **🎲 Free Bet** — split 90/10 automatico

## Deploy su GitHub Pages (3 passi)

1. Crea repository pubblico su GitHub
2. Carica tutti e 4 i file: `index.html`, `manifest.json`, `sw.js`, `icon.svg`
3. **Settings → Pages → Source: main branch / root** → Save

Disponibile su `https://tuousername.github.io/nomerepo/`

## Installa come App

- **iPhone/iPad**: apri con **Safari** → Condividi → **"Aggiungi a schermata Home"**
  *(deve essere aperto con Safari, non Chrome!)*
- **Android**: Chrome → ⋮ → "Aggiungi a schermata Home"
- **Desktop**: icona installa nella barra indirizzi

## Note iOS

Su iPhone aprire SEMPRE con Safari (non Chrome) per poterla aggiungere alla schermata home.
Una volta installata funziona offline grazie alla cache.
